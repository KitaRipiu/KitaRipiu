1. Instalasi
	*Install Node JS
	*Install Meteor : curl https://install.meteor.com/ | sh
	*Clone Repository
	*Checkout branch develop : git checkout develop
	*jalankan npm install
	
2. Menjalankan Aplikasi
	*jalankan perintah "meteor" untuk Run aplikasi.
	*Buka Localhost:3000
